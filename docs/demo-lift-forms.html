üöÄ Getting Started

Admin Access Required: You need admin privileges to access the Form Builder. The LIFT Forms feature is integrated into your LIFT Docs admin area.

Step 1: Access Form Builder

Navigate to your WordPress admin and look for:

LIFT Documents ‚Üí Forms ‚Üí Form Builder

Step 2: Create Your First Form

  1. Enter a form name and description
  2. Drag fields from the left panel to the canvas
  3. Click on fields to customize their settings
  4. Organize fields using columns and sections
  5. Save your form

Step 3: Display Your Form

Use the shortcode to display your form on any page or post:

[lift_form id="1" title="true"]

Step 4: Manage Submissions

View and manage form submissions at:

LIFT Documents ‚Üí Forms ‚Üí Submissions

üí° Tips & Best Practices

Form Design Tips

  • Keep forms short and focused
  • Use clear, descriptive field labels
  • Group related fields in sections
  • Provide helpful placeholder text
  • Mark required fields clearly

User Experience

  • Test forms on mobile devices
  • Use appropriate input types
  • Provide clear error messages
  • Show progress for long forms
  • Confirm successful submissions

Performance

  • Optimize file upload sizes
  • Use pagination for very long forms
  • Enable caching where appropriate
  • Monitor submission volumes
  • Regular database cleanup

üÜò Troubleshooting

Common Issues

Form Not Displaying

  • Check form status is "Active"
  • Verify shortcode ID is correct
  • Ensure form has fields added
  • Check for plugin conflicts

Submissions Not Working

  • Check AJAX functionality
  • Verify nonce validation
  • Review server error logs
  • Test with default theme

Debug Mode

Enable WordPress debug mode to see detailed error information:
